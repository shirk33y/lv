# Windows x64 runtime DLLs

- **SDL2.dll** — from [libsdl.org](https://github.com/libsdl-org/SDL/releases) v2.30.12
- **libmpv-2.dll** — fetched at build time from [shinchiro builds](https://sourceforge.net/projects/mpv-player-windows/files/libmpv/) (112MB, too large for git)

Run `../pkg/fetch-mpv.sh` to download libmpv-2.dll into this directory.
